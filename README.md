📄 Resume Builder – Full Stack Project

A full-stack Resume Builder web application that allows users to create professional resumes easily using predefined templates.

Users can:

- Fill their personal and professional details
- Preview the resume in real time
- Download the resume as a PDF
- Share the resume through email
- The system also includes premium resume templates that can be unlocked through online payment integration.

🚀 Project Features

🔐 User Authentication

- User Signup and Login
- Email verification during registration
- Profile photo upload during signup
- Secure authentication system

📝 Resume Builder

- Multiple professional resume templates
- Live preview while entering details
- Easy and user-friendly interface
- Properly structured resume layout

📄 Resume Export

Users can:
- Download resume as PDF
- Send resume via email directly from the platform

💎 Premium Templates
- Two free templates
- Two premium templates
- Premium templates can be unlocked through payment integration
- Payment gateway integrated using Razorpay

⚙️ Installation & Setup

1️⃣ Clone the Repository

git clone https://github.com/yourusername/Resume-Builder-Full-Stack-Project.git

2️⃣ Backend Setup (Spring Boot)

1. Open backend project in IntelliJ / Eclipse

2. Configure database in application.properties

spring.datasource.url=jdbc:mysql://localhost:3306/resume_builder
spring.datasource.username=root
spring.datasource.password=yourpassword

3. Run the Spring Boot Application

3️⃣ Frontend Setup (React)

- Navigate to frontend folder:
cd Project Frontend/resume-builder

- Install dependencies:
npm install

- Start the application:
npm start
